# Alabaster Themes for Ghostty

## Installation

Copy the theme files to your Ghostty themes directory:

```bash
mkdir -p ~/.config/ghostty/themes
cp Alabaster-Dark Alabaster-Light ~/.config/ghostty/themes/
```

Then add to your `~/.config/ghostty/config`:

```
theme = Alabaster-Dark
```

Or for light mode:

```
theme = Alabaster-Light
```

## Files

- `Alabaster-Dark` - Dark theme (no file extension)
- `Alabaster-Light` - Light theme (no file extension)

## Notes

Ghostty theme files have no extension by convention. The filename becomes the theme name.
