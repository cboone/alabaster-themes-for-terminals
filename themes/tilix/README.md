# Alabaster Themes for Tilix

## Installation

Copy the `.json` files to the Tilix schemes directory:

```bash
mkdir -p ~/.config/tilix/schemes
cp *.json ~/.config/tilix/schemes/
```

Then in Tilix:

1. Go to **Preferences**
2. Select your profile
3. Go to the **Color** tab
4. Select the Alabaster theme from the **Color scheme** dropdown

## Files

- `Alabaster-Dark.json` - Dark theme
- `Alabaster-Light.json` - Light theme
