# Alabaster Themes for iTerm2

## Installation

1. Open iTerm2
2. Go to **Preferences** (Cmd+,)
3. Select the **Profiles** tab
4. Select the **Colors** subtab
5. Click **Color Presets...** dropdown
6. Select **Import...**
7. Choose the `.itermcolors` file
8. Click **Color Presets...** again and select the imported theme

## Files

- `Alabaster-Dark.itermcolors` - Dark theme
- `Alabaster-Light.itermcolors` - Light theme
