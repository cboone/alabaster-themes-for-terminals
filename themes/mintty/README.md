# Alabaster Themes for mintty

## Installation

Copy the `.minttyrc` files to the mintty themes directory:

- Windows: `%APPDATA%\mintty\themes\`
- Cygwin/MSYS2: `~/.config/mintty/themes/`

Then in mintty:

1. Right-click the title bar and select **Options**
2. Go to the **Looks** section
3. Click **Theme...** and select the Alabaster theme

Alternatively, add to your `~/.minttyrc`:

```
ThemeFile=Alabaster-Dark
```

## Files

- `Alabaster-Dark.minttyrc` - Dark theme
- `Alabaster-Light.minttyrc` - Light theme
