# Alabaster Themes for Warp

## Installation

Copy the theme files to your Warp themes directory:

**macOS:**
```bash
mkdir -p ~/.warp/themes
cp *.yaml ~/.warp/themes/
```

**Linux:**
```bash
mkdir -p ~/.local/share/warp-terminal/themes
cp *.yaml ~/.local/share/warp-terminal/themes/
```

Then restart Warp and select the theme from:

**Settings** > **Appearance** > **Themes**

## Files

- `Alabaster-Dark.yaml` - Dark theme
- `Alabaster-Light.yaml` - Light theme
