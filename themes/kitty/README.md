# Alabaster Themes for Kitty

## Installation

### Option 1: Include directive

Copy the theme file to `~/.config/kitty/themes/` and add to your `~/.config/kitty/kitty.conf`:

```
include themes/Alabaster-Dark.conf
```

### Option 2: Direct copy

Copy the contents of the theme file directly into your `~/.config/kitty/kitty.conf`.

### Option 3: Kitty themes command

```bash
kitty +kitten themes
```

## Files

- `Alabaster-Dark.conf` - Dark theme
- `Alabaster-Light.conf` - Light theme
