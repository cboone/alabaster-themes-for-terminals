# Alabaster Themes for Konsole

## Installation

Copy the `.colorscheme` files to one of these locations:

- User directory: `~/.local/share/konsole/`
- System directory: `/usr/share/konsole/`

Then in Konsole:

1. Go to **Settings** > **Edit Current Profile...**
2. Select the **Appearance** tab
3. Choose the Alabaster theme from the list

## Files

- `Alabaster-Dark.colorscheme` - Dark theme
- `Alabaster-Light.colorscheme` - Light theme
